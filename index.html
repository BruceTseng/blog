SASS 變數 - 常用語法一鍵設定

Variable(變數)
為什麼要用變數？

  • 參數統一管理，若使用相同變數過多，可以同時修正內容
  • 語意化
  • 將時常需要設定的CSS放在變數來管理
  • 變數是用美元符來命名

Example

常見的錯誤

  • 宣告每一個屬性後都要用;結束

  • CSS編譯有順序性，因此變數要放前面

變數格式
格式

  • 數字：10、1px、3em
  • 字串：’test’、’../images/big/‘、字體
  • 顏色：red、#000000、rgba(255, 255, 255, 0.9)
  • 布林：true、false
  • 空值：null
  • 支援運算(1em+1em、6px*2)

藉由 darken、lighten內建功能調整明暗

SMACSS

結構分類與命名規則

  • 結構分類：Base、Layout、Module、State、Theme。
  • 命名規則：id 與 class 受限制地使用、名稱使用 dash 分隔。

結構

  • Base：全站設定, CSS Resets。

  • Layout：網頁布局，例如：header, footer

    Generally, a Layout style only has a single selector: a single ID or class name.

  • Module：增加子模組，用class方式命名並用dash分隔，例如 .btn-primary。

方便HTML語意化調整，盡量不指定HTML tag
Example

  • State：描述元件的狀態，例如：tab-item active 的 active。
  • Theme：針對網站主視覺而定義的 Layout 或 Module 的樣式。 例如 .tab-dark。

OOCSS

  • 結構與樣式分離
